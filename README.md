# PhonIQ — 英語発音記号マスター

英語の発音記号（IPA）を体系的に学習するWebアプリです。

## 機能

### フェーズ1：理論学習
- 発音記号を学ぶ意義の解説（4カード）
- 44音素を9カテゴリに分類したカタログ
  - 短母音 / 長母音 / 二重母音 / 破裂音 / 摩擦音 / 鼻音 / 接近音 / 破擦音
- 各音素クリックで詳細パネル表示（口の形・例単語・類似音素との違い）
- Web Speech APIによる音声再生
- IPAトラペゾイド（母音チャート）の可視化

### フェーズ2：問題実践
- 難易度3段階（初級・中級・上級）
- 1問につき3つのサブ設問：
  - A. アクセントのある音節はどれか（音節ごとに分かれた表示）
  - B. 強勢音節の発音記号は何か（4択）
  - C. 最も発音が近い単語はどれか（4択）
- 連続正解ストリーク表示
- セット終了後のスコアレポート

## 技術スタック

- 純粋なHTML/CSS/JavaScript（依存ライブラリなし）
- Web Speech API（発音音声）
- IPA国際音声記号データベース（44音素）
- Nexus Design System（ダーク/ライトモード対応）

## 使い方

`phoniq.html` をブラウザで開くだけで動作します。
